¿Que es? 
Es un sitio web con EXPERIENCIA DE USUARIO de una aplicación mobile.
    > Segura (HTTPS) 
    > Instalable (ícono en el escritorio)
    > Offline


¿Para que sirve?
Cuando la vista mobile no necesita características de una aplicación mobile nativa.
Ventajas
    * Es barata
        - tiempo
        - costo
        - aprendizaje
        - cambios
Desventajas
    * No cuenta con todas las características de una aplicación mobile (acceso a contactos, persistencia)
    * No se puede publicar en el Store.
    * Tiene una actualización más dificultosa.


¿Cómo se usa? ¿Cómo se aprende?
* Concepto de Proxy de Red (ServiceWorker)
